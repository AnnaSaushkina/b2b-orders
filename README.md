## Проект: B2B платформа управления дистрибьюцией

**Стек:** React · TypeScript · NestJS · Prisma · PostgreSQL · FSD

---

## Реализовано [24.06]:

Виртуализация списка товаров без библиотек — 12 DOM-узлов вместо 200 при 50 000+ товарах в БД.

### Метрики

|                   | DOM-узлов |
| ----------------- | --------- |
| Без виртуализации | 50000     |
| С виртуализацией  | ~12       |
